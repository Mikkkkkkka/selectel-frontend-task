# Selectel Frontend Task

Мини-приложение на Angular 19+ по тестовому заданию.

## Требования задания
1. Две страницы: приветствие и меню.
2. На странице меню — список элементов с checkbox и значениями.
3. В заголовке меню показываются: выбранное меню, количество выбранных элементов и их сумма.
4. Данные статичны, backend не нужен.

## Реализовано
- Страница `Home` с приветствием и ссылкой на меню.
- Страница `Menu` со списком чекбоксов и значений.
- Заголовок меню с количеством выбранных и суммой.
- Статичные данные в компоненте.

## Технологии
- Angular 21.2
- TypeScript
- HTML/CSS

## Запуск локально
```bash
npm install
ng serve
```
Открыть `http://localhost:4200/`.

## Сборка
```bash
ng build
```
Артефакты будут в `dist/`.

## Структура проекта
- `src/app/pages/home` — страница приветствия
- `src/app/pages/menu` — страница меню
- `src/app/app.routes.ts` — маршрутизация
